Unannounced 2D Beat-em-up Game

Unannounced 2D Beat-em-up Game
(WayForward Technologies)

A 2D sidescrolling "beat-em-up" styled game. Players select their loadout, class, which quests the're trying to complete, and world augmentations before starting a run. Features online multiplayer where players can play together.

Developed in Unity Engine written in C#. I did primarily front-end development, developing all of the UI including in-game UI, in-game menus, title screen, and hub menus. Ultimately, building most features into the main hub screen which interfaces with every feature within the game.
