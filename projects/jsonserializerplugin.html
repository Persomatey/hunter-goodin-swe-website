Unity JSON Serializer Plugin

Unity JSON Serializer Package

A JSON serializer package for the Unity game engine. Users can read from and write to .json files provided as Unity TextAsset variables. Users can read and change the values of specific variables from the JSON object without needing to create custom classes for each .json file. Users can also add new variables to the file in code using Add functions.

Made in the Unity Engine written in C#. I solo developed the package, doing all of the programming.
